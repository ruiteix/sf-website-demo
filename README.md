Requirement
-----------

* Docker & docker compose
* Unix like system

Installation
------------

* Initialize

```make init```

Make sure to personalize the docker-compose.override.yml

* Pull, build & start containers

```make up```

* Install dependances

```make install```

* Go to http://demo.docker.localhost/
